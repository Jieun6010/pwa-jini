css Cascading Style Sheet

우선순위와 상속이 존재하는 디자인 양식

type 선택자(태그선택자)

사용법

클래스명{속성:값} 형태로 선언

특징

  • 동일한 태그에 모두 적용됨
  • 자식태그선택자가 우선순위가 높다
  • agent style과 겹칠(충돌) 수 있다

class 선택자

사용법

.클래스명{속성:속성값;}을 이용해서 선언 후 class속성을 이용해서 호출/ .뒤에 숫자가 들어가서는 안됨

특징

  • 타입선택자보다 우선순위가 높다
  • 반복호출이 가능하다
  • 수정관리가 용이하다. 한번에 묶어서 바꿀 수 있기 때문에

id선택자

사용법

#아이디명{속성:속성값} 선언 후, id속성을 이용해서 호출

특징

  • id는 class보다 우선순위가 높다.
  • 반복호출이 허락되지 않고 수정관리가 용이하지않아서 디자인목적으로는 거의 쓰지 않는다.
  • html문법이나 프로그램용도로 사용

inline 스타일

사용법

요소의 style속성을 지정 후 css를 작성

한 줄로 그냥 쫘아악 쓰는 것

특징

  • 우선순위가 id보다도 높다.
  • 쓸 일이 거의 없지만, 자바스크립트를 이용해 css를 제어하는데 사용

css의 특징

  • 요소단위로 css가 적용됨
  • 우선순위 1.inline style, 2.id, 3.class, 4.자식태그, 5.부모태그
  • 글자정렬, 글자관련 스타일은 상속이 된다.
