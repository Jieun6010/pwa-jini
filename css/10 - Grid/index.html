grid system

  • display속성이 grid인 컨테이너는 block레벨과 유사하다.
    grid 컨테이너의 자식요소는 기존의 display속성이 사라진다. 마진오토 X
  • grid컨테이너의 격자 설정
    display: grid; grid-template-rows : repeat(갯수, 한 줄의 높이(auto, 1fr)) grid-template-columns : repeat(갯수, 한 칸의 높이(auto, 1fr)) gap: 그리드간의 여백 설정
  • grid를 설정
    grid-row: 2 / 6 (2번째 줄에서 시작해서 5번째 줄에서 끝난다.)
    grid-columns: 2 / span 3 (2번째 칸에서 시작하는 3칸짜리)
    grid-area: 2 / 3 / 7 / 4
    (2번째 줄에서 시작해서 6번째 줄에서 끝나면서
    3번째 칸에서 시작해서 3번째 칸에서 끝난다.)
