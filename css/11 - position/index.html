position

좌표를 이용해서 위치를 설정

  • abosolute

    • 기존의 display 속성이 적용되지 않는다.
    • 좌표를 이용해서 독립적인 위치로 이동한다.
    • 좌표 : (left, right) (top, bottom)
    • 좌표는 position속성을 가지고 있는 상위요소 기준
    • z-index가 적용됨
    • position이 적용된 자식의 크기를 부모요소는 인식할 수 없다. / 좌표의 기준점이 부모몸통이다.

  • fixed

    좌표의 기준점이 브라우저다. (스크롤의 위치 영향을 받지 않는다.) 을 제외하면 abs와 동일함

  • relative

    • 현재 레이아웃을 유지하며 position속성을 추가할 수 있다.
    • 초기위치(태어난위치, 좌표를 지정하기 전의 위치)를 좌표의 0점으로 설정한다.
