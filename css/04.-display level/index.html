출력레벨(방식)

block level

  • 한 줄을 통째로 차지하는 출력요소
  • 크기속성을 가질 수 있다.(width/higth)

inline level

  • 한줄로 출력되는 요소
  • 크기속성이 적용되지 않는다.
  • a,span,small,em,strong,i,label 등등~
span bold

liline-block 무조껀 암기하자!!

  • 한줄로 출력되는 요소
  • 크기 속성이 적용됨
  • main,section/article,header,footer,nav,div, ul,ol,dl,li,dt,dd, h1~h6,p,form,table 등등~
  • img,input,select,textarea,iframe,svg,video 얘들만 알아도 코딩하는데 큰 어려움 없음

디스플레이레벨과 정렬

인라인레벨의 수직, 수평정렬

  • vertical-alingn ( top , bottom , midle ) ( 인라인계열만 적용되는 속성 )
  • 엘리먼트 안쪽의 글자(인라인계열)을 중앙정렬하기: 줄간격(line-height)을 높이와 동일하게 설정(한 줄일 경우만 사용가능)
  • 글자(인라인계열)을 수평정렬 할 경우 부모요소에서 text-align 속성을 이용한다.
