변형과 모션

transform

  • 레이아웃에는 변화가 없는 상태에서 렌더링에 변화를 준다.
  • transform: scale(2) rotate:(45deg) translate:(x,y)이렇게 계속 이어붙일 수 있다.
  • transform, opacity, position 프로퍼티가 적용되면 z-index가 적용돼서 사용할 수 있게됨.
  • transform의 %기준은 해당 요소의 크기 기준이다.

transition () 변환효과

  • css 프로퍼티가 변하는 시간을 설정하는 기능이다.
  • transition:프로퍼티 || all 시간s
  • transition은 숫자단위로 설정이 가능한 속성에만 적용가능하다.
  • transition-delay : 프로퍼티간의 transition 시작 시간 지연
  • transition-delay효과를 구현하려면 all을 사용하지 말 것
    귀찮더라도 하나씩 별도로 지정을 해줘야한다.
