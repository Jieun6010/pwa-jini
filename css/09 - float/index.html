float을 이용한 그리드 레이아웃

다양한 레이아웃 구현목적

  • float이 적용된 자식객체의 높이를 부모객체가 인식하지 못함
    (예 / 엄마가 임신했는데 아이는 배 밖에 있어서 엄마 배는 홀쭉!)
    clearfix를 이용해 해결할 수 있음.
     
        부모요소::after{
          content:""; 
          display:block;
          clear:both;
        }
  • float이 적용되면 기존의 display 속성을 잃는다.
    margin의 auto값을 사용할 수 없음.
    inline 요소도 크기를 지정할 수 있다.
  • float이 적용된 다음요소는 float과 clear(형이랑은 관ㄱㅖ를 끊겠다! 따로서겠다!) 중 반드시 하나를 택일해야 한다.
  • float레이아웃 시 margin-bottom을 사용 권장
    (float이 적용되지 않은 객체는
    윗쪽에 위치하는 float요소와 margin이 적용되지 않는다.)
